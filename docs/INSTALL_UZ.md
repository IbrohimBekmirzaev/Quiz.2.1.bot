## 2. Terminal

npm install
```

## 3. .env tayyorlang

- TOPIC_QUIZ_ID
- TOPIC_LINK_ID
- TOPIC_SUPPORT_ID
- TOPIC_ERROR_ID
- TOPIC_USERS_ID (ixtiyoriy, alohida users topic uchun)
- SECOND_TOPIC_START_ID (ixtiyoriy)
- SECOND_TOPIC_QUIZ_ID (ixtiyoriy)
- SECOND_TOPIC_LINK_ID (ixtiyoriy)
- SECOND_TOPIC_SUPPORT_ID (ixtiyoriy)
- SECOND_TOPIC_ERROR_ID (ixtiyoriy)

- BOOKS_API_URL
- API_URL

npm start
```
